## moyuRead

这是一个在编写代码的过程中，发出 valorant 击杀音效的插件

### 触发条件

|    支持的操作    |
| :--------------: |
| delete\enter\tab |

-   按下上面的按键就会触发音效，音效从一杀开始增加，直到重置
-   **批量操作不会触发音效**

### 使用

-   读取文本
    `ctrl+shift+p`:显示所有命令
    输入`reader:add-text`确定
